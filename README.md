# Self-Diagnosis and Large Language Models: A New Front for Medical Misinformation

The code provided in this repository acts as a sample of the processing completed for access the OpenAI API for prompt processing.

The foundation for the question database used within this project is obtained from ''Performance of ChatGPT on USMLE: Potential for AI-assisted medical education using large language models'', by Kung et al. at https://doi.org/10.1371/journal.pdig.0000198.s001.
